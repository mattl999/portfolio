Summer Heights Band Website!

This project is the official website for Toronto-based pop-punk band 'Summer Heights'.
It is my first freelance project for a client, and I had a lot of fun making it! The project consists of a Bootstrap/EJS frontend, and a small Node.js/MongoDB backend, which allows the client to update their tour dates on the go!
This project forced me out of my comfort zone, and I enjoyed learning new technologies, as well as client communication skills along the way!
The app is currently running from a docker container hosted on a Lightsail VPS instance.
