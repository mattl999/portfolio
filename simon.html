Simon in Space!

Simon is a fun and engaging game, with the benefit of testing and improving your memory while you play! The game requires lots of focus, and the smallest distraction can prevent you from reaching your new highscore. The current world record in a game of Simon is 84, set by a Canadian Simon enthusiast! This application is a clone of the classic game, except it is set in space!
